# ことりのゆりかご 🐤

見ているだけで癒される、ふしぎなことりアプリ。
スマホのブラウザで開くだけで遊べます。サーバ不要・単一HTMLファイル。

**公開URL(GitHub Pages設定後):** https://miharubajp.github.io/daydream/

## あそびかた

- まるっとした「もちもち鳥」たちが、空から遊びにやってきます
- **じめんをタップ** → たねをまく 🌱 → 鳥たちがとことこ集まってついばみます
- **鳥をタップ** → ぷにぷにむにっと潰れて「ぴよっ」と鳴きます
- たねを食べたり触れあったりすると **なつき度** が上がり、レベルアップすると
  新しい種類の鳥(白いの、青いの、ピンクの、金色のレアな子…)が遊びに来ます
- たね(最大10個)は実時間で回復します。アプリを閉じていてもたまるので、
  なんとなくまた開きたくなります
- ときどき **ぬすっとバト**(憎たらしいけどどこか可愛い太った鳩)が現れて
  たねを盗んでいきます。3回つつくとぷくーっと膨れてたねを返して逃げていき、
  たまにレアな羽根を落としてくれます
- 空の色は**実際の時刻**で変わります。夜は星が瞬き、流れ星が流れることも 🌙
- 右上の 🕊️ ボタンから、なつき度をそのままツイートで自慢できます
- 効果音はすべてWeb Audioでその場で合成(音声ファイルなし)。🔊 ボタンでミュートできます
- 進行状況(なつき度・たね・出会った鳥)はブラウザのlocalStorageに保存されます

減点も空腹もゲームオーバーもありません。完全に平和です。

## 公開手順(初回のみ)

1. このリポジトリの `main` ブランチにマージする
2. GitHub のリポジトリページで **Settings → Pages → Build and deployment → Source** を
   **「GitHub Actions」** に変更する
3. `main` へのpushをトリガーに `.github/workflows/deploy-pages.yml` が自動でデプロイします
   (初回はマージ後に Actions タブから `Deploy to GitHub Pages` を手動実行してもOK)
4. 数十秒で https://miharubajp.github.io/daydream/ に公開されます。URLをツイートして共有 🐦

## 開発メモ

- `index.html` 1ファイルにすべて(HTML/CSS/JS)が入っています。依存ライブラリなし
- 描画はCanvas 2D、鳥も背景もすべてプロシージャル描画(画像アセットなし)
- デバッグ用URLパラメータ: `?t=22` で時刻を22時に固定(空の見た目確認用)
- テスト用フック: DevToolsで `window.__kotori` から状態の確認・たねまき・ハト召喚ができます
